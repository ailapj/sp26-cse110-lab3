# Styling Meeting Minutes

## Comment
- Use comments in CSS for better readability and maintainability.

## Color
- Use consistent color palette.
- Prefer variables for reusable colors.

## CSS Variables & Fallbacks
- Define variables using `:root`
- Provide fallback values where necessary.

## Background
- Use background-color, gradients, or images appropriately.

## Unit
- Prefer relative units like rem, em, % over px.

## Box Model
- Understand margin, padding, border, content.
- Use `box-sizing: border-box`.

## Text
- Maintain readable font size and line height.
- Ensure good contrast.

## Display
- Use block, inline, flex, grid as needed.

## Sizing
- Use responsive sizing techniques.

## Position
- Understand static, relative, absolute, fixed, sticky.

## Pseudo-class
- Use :hover, :focus, :active for interactions.

## Layouts
- Use Flexbox and Grid for modern layouts.

## Responsiveness
- Use media queries.
- Mobile-first approach preferred.

## Fonts
- Use web-safe or imported fonts.
- Maintain consistency across UI.
